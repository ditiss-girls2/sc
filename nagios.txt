========================= NCPA INSTALLATION ================================

-------------------------
Ubuntu
-------------------------

Download

wget https://assets.nagios.com/downloads/ncpa/ncpa-latest-ubuntu-amd64.deb

Install

sudo dpkg -i ncpa-latest-ubuntu-amd64.deb

If dependency error

sudo apt --fix-broken install -y

Install Again

sudo dpkg -i ncpa-latest-ubuntu-amd64.deb

Start Service

sudo systemctl enable ncpa_listener

sudo systemctl start ncpa_listener

Check Status

sudo systemctl status ncpa_listener



-------------------------
CentOS 9
-------------------------

Download

wget https://assets.nagios.com/downloads/ncpa/ncpa-latest.el9.x86_64.rpm

Install

sudo dnf install ./ncpa-latest.el9.x86_64.rpm

Start Service

sudo systemctl enable ncpa_listener

sudo systemctl start ncpa_listener

Check Status

sudo systemctl status ncpa_listener



====================== HOW TO FIND NCPA TOKEN ==============================

Method 1 (Recommended)

sudo cat /usr/local/ncpa/etc/ncpa.cfg

OR

sudo nano /usr/local/ncpa/etc/ncpa.cfg

Find

community_string = mytoken

Example

community_string = cdac123

Token = cdac123



Method 2

grep community_string /usr/local/ncpa/etc/ncpa.cfg



==================== CHANGE TOKEN =========================================

sudo nano /usr/local/ncpa/etc/ncpa.cfg

Change

community_string = mytoken

to

community_string = cdac123

Save

Ctrl + O
Enter
Ctrl + X

Restart

sudo systemctl restart ncpa_listener



====================== VERIFY NCPA ========================================

Check Service

systemctl status ncpa_listener

Expected

Active (running)



Check Port

ss -tulnp | grep 5693

Expected

5693 LISTEN



Check Browser

https://<Host_IP>:5693



Check API

https://<Host_IP>:5693/api?token=<community_string>

Example

https://192.168.80.138:5693/api?token=cdac123

Expected

JSON Output
